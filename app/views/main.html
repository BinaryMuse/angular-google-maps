Google Maps for Angular.js

Simple to use Angular.js directive to work with Google Maps inside your applications.

Download View on Github

Features

  • Two-way binding of center, zoom & marker properties;
  • Support for polylines

Headlines

Working on polyline integration
We're currently working to the integration of polylines. This will introduce a new directive, polyline, which will be used inside the google-map directive to draw a line between a list of coordinates.
